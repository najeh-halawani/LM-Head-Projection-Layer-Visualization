↻ for sb = 0 … 3

↻ for g = 0 … 1 // rows 0–3 / 4–7

reconstruct 6-bit weights → 2 × SDOT (low & high runs)

acc[g] −= bias[g]  (apply −32)  ·  logit += float(acc[g]) · d_q6 · d_q8
